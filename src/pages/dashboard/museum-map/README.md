# 博物馆地图分布页面

## 功能概述

这是一个基于ECharts的交互式中国博物馆分布地图，支持省级和市级的下钻功能，提供直观的数据可视化体验。

## 主要功能

### 🗺️ 地图展示
- **中国地图全景**：默认显示中国地图，展示各省份博物馆数量分布
- **省级下钻**：点击省份可下钻到该省的城市分布地图
- **市级详情**：点击城市可查看该城市的博物馆列表

### 📊 数据统计
- **统计面板**：显示总博物馆数量、覆盖区域、开放博物馆数量和开放率
- **动态更新**：根据当前地图层级动态更新统计数据
- **可视化映射**：使用颜色深浅表示博物馆数量多少

### 🔍 交互功能
- **鼠标悬停**：显示当前区域博物馆数量信息
- **点击下钻**：支持从国家→省份→城市的层级下钻
- **返回导航**：提供面包屑导航和返回上级功能
- **博物馆列表**：抽屉展示选中城市的博物馆详细列表

### 📱 博物馆详情
- **列表展示**：表格形式展示博物馆基本信息
- **详情跳转**：支持跳转到博物馆详情页面
- **状态标识**：显示博物馆等级、开放状态等信息

## 技术实现

### 核心技术栈
- **React 18**：现代化的React Hooks组件
- **ECharts 5**：强大的数据可视化库
- **echarts-for-react**：React的ECharts封装组件
- **Ant Design Pro**：企业级UI组件库


### 数据流
1. **初始化**：加载中国地图数据和省份博物馆统计
2. **省级下钻**：点击省份加载城市地图数据
3. **城市详情**：点击城市获取博物馆列表数据
4. **详情跳转**：跳转到博物馆管理页面查看详细信息

## 使用方式

### 基本操作
1. **查看全国分布**：页面默认显示中国地图全景
2. **省份下钻**：点击任意省份查看该省城市分布
3. **城市详情**：点击城市名称查看博物馆列表
4. **返回上级**：使用导航栏的返回按钮
5. **查看详情**：点击博物馆名称跳转详情页

### 交互提示
- 🖱️ **鼠标悬停**：查看区域博物馆数量
- 🖱️ **单击**：下钻到下一级或显示详情
- 📊 **统计面板**：实时显示当前层级数据
- 🧭 **面包屑**：显示当前位置和层级关系

## 扩展功能

### 可扩展特性
- **更多地图层级**：可扩展到区县级别
- **实时数据**：支持接入实时博物馆数据API
- **数据筛选**：支持按博物馆类型、等级筛选
- **导出功能**：支持地图截图和数据导出

### 自定义配置
- **地图样式**：可自定义地图配色方案
- **统计维度**：可添加更多统计指标
- **交互方式**：可自定义点击和悬停行为

## 注意事项

1. **地图数据**：使用 `echarts-maps` 包提供的标准中国地图数据，包含完整的省份边界信息
2. **省份地图**：目前支持北京、福建、河北等省份的下钻地图，其他省份显示城市列表
3. **API集成**：已预留博物馆数据API接口，支持按省份和城市筛选
4. **性能优化**：大数据量时建议使用虚拟化或分页加载
5. **移动端适配**：建议针对移动端进行交互优化
